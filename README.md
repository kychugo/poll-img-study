

# Pollination Image Generator Hub

The Pollination Image Generator Hub is a user-friendly, open-source web platform designed to transform educational topics into engaging, comic-style visuals. It is built as a single-page HTML application that leverages the power of the [Pollinations.ai API](https://pollinations.ai/) to create unique study aids.

This tool is specifically focused on making learning more dynamic and memorable. Instead of being a general-purpose image generator, its primary feature, "Study Comics Mode," uses a specialized prompt to break down complex subjects into a fun, manga-panel format, making it an ideal companion for students, educators, and lifelong learners.

## How to Use

This platform is a single HTML file, making it incredibly easy to set up and run.

1.  **Get an API Key**: Sign up at [enter.pollinations.ai](https://enter.pollinations.ai) to get your free or paid API key.
2.  **Download**: Download the `index.html` file from this repository.
3.  **Open in Browser**: Open the downloaded `index.html` file in any modern web browser (like Chrome, Firefox, or Edge).
4.  **Set Your API Key**:
    *   Click the settings icon (âš™ï¸).
    *   Enter your Pollinations API Key in the designated field. The key will be saved in your browser for future use.
5.  **Generate!**
    *   Select **"Study Comics Mode"** and enter a topic in the text box.
    *   Alternatively, select **"Custom Mode"** to write your own detailed prompt.
    *   Click the **"Generate"** button and wait for your image to appear.
